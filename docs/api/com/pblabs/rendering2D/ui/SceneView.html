Classpublic class SceneView
InheritanceSceneView flash.display.Sprite
Implements IUITarget

This class can be set as the SceneView on the BaseSceneComponent class and is used as the canvas to draw the objects that make up the scene. It defaults to the size of the stage.

Currently this is just a stub, and exists for clarity and potential expandability in the future.
