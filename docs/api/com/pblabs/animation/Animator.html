Packagecom.pblabs.animation
Classpublic class Animator
InheritanceAnimator flash.events.EventDispatcher
Subclasses FrameAnimator, PointAnimator

Class for animating between a start value and end value.

To animate more complex types, subclass this and implement the interpolate method.

interpolate()method 
protected function interpolate(start:*, end:*, time:Number):*

Performs the actual animation. This can be overridden by subclasses to interpolate more complex types. This default implementation will only work for values that can use the +, -, and operators.

Parameters

start:* — The value to interpolate from.
 
end:* — The value to interpolate to.
 
time:Number — The interpolation factor. A value of 0 will return the start value, a value of 1 will return the end value. A value of 0.5 should return the value half way between start and end.

Returns
* — The interpolated value.
