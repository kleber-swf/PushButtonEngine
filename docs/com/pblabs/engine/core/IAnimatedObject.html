Interfacepublic interface IAnimatedObject
Implementors AnimatedComponent, LogViewer, ScreenManager

This interface should be implemented by objects that need to perform actions every frame. This is most often things directly related to rendering, such as advancing frames of a sprite animation. For performing physics, processing AI, or other things of that nature, responding to ticks would be more appropriate.

Along with implementing this interface, the object needs to be added to the ProcessManager via the AddAnimatedObject method.
