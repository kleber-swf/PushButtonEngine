Packagecom.pblabs.engine.debug
Classpublic class LogEntry
InheritanceLogEntry Object

Log entries are automatically created by the various methods on the Logger class to store information related to an entry in the log. They are also dispatched in the LogEvent when the entry is added to the log to pass information about the entry to the listener.
