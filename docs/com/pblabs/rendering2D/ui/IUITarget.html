Interfacepublic interface IUITarget extends flash.events.IEventDispatcher

Interface a UI element must implement to work with the DisplayObjectScene or its subclasses. Deals with adding/removing display objects safely. Only one scene can draw to a IUITarget at a time (and a IUITarget can only be used by one scene at a time).
