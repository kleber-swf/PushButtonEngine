Packagecom.ffcreations.ui.mouse
Interfacepublic interface IMouseInputComponent extends IEntityComponent
Implementors GUIComponent, MouseInputComponent

Interface that must be implemented to handle with mouse inputs. When implementing this interface, you should add the component to PBE.mouseInputManager to make it renpond to mouse events and remove it from the same class when it's not used anymore. When updating its priority, the MouseInputManager must be informed too.
